# 3D Helicopter & Interactive Scene Simulator

這是一個基於 **OpenGL (GLUT)** 開發的互動式 3D 場景模擬程式。本專案透過層級建模（Hierarchical Modeling）技術構建了一架具備完整動畫邏輯的直升機，並在一個包含城堡、動態光照、日夜循環與多種視覺特效的 3D 世界中進行展示。

## 核心技術亮點

### 1. 層級建模與動畫 (Hierarchical Modeling)
* **直升機系統**：利用 `glPushMatrix` 與 `glPopMatrix` 實作機身、旋轉螺旋槳、起落架輪胎的層級關聯。
* **起降邏輯**：實作自動化的起飛（傾斜滑行）與降落序列。
* **動態環境**：包含可旋轉的熊模型、會擺動的公園鞦韆，以及具備發光窗戶的高塔。

### 2. 進階光照與環境系統
* **日夜循環 (Day/Night Cycle)**：按下 `Space` 鍵可即時切換清晨、正午、黃昏與深夜，天空球顏色與環境霧氣會隨之聯動。
* **多光源應用**：
    * **Spotlight (聚光燈)**：直升機探照燈（支援顏色切換與照射範圍調整）與動態旋轉舞台燈。
    * **Point Light (點光源)**：具備「呼吸燈」模式的路燈系統。
    * **Fireworks (煙火)**：實作粒子爬升、高空綻放與隨機放射的視覺效果。
* **霧氣效果 (Fog)**：支援 `Linear`、`Exp`、`Exp2` 三種模式，並可手動調整濃度。

### 3. 視覺特效技術
* **紋理貼圖 (Texture Mapping)**：使用 `stb_image` 讀取 PNG/JPG，應用於天空、建築、地磚與角色。
* **廣告板技術 (Billboard)**：實作機車與 Padoru 角色，確保 2D 貼圖在 3D 空間中永遠面向觀察者。
* **多視窗展示 (Viewport)**：提供五種視角模式，支援同時觀看正交投影（三視圖）與透視投影視角。

## 操作指南

### 飛行與動作控制
| 按鍵 | 功能描述 |
| :--- | :--- |
| **L / l** | **起飛 / 降落** (包含自動化姿態調整) |
| **W / S** | 直升機前進 / 後退 |
| **A / D** | 直升機水平左轉 / 右轉 |
| **Q / E** | 直升機下降 / 上升 |
| **O / P** | 公轉模式 (繞中心左旋 / 右旋) |
| **B / b** | 熊熊互動 (坐起躺下 / 水平旋轉) |

### 環境與光照互動
| 按鍵 | 功能描述 |
| :--- | :--- |
| **Space** | 切換**日夜循環**狀態 |
| **0 (零)** | 施放**高空煙火** |
| **. / ,** | 路燈開關 / 切換模式 (常亮 vs 呼吸燈) |
| **[ / ]** | 調整直升機探照燈的照射廣度 (Cutoff) |
| **- / =** | 切換探照燈顏色 / 旋轉探照燈方向 |
| **M / m** | 切換霧氣模式 (OFF / Linear / Exp / Exp2) |
| **; / '** | 霧氣濃度 減少 / 增加 |
| **/ (斜線)** | 舞台燈開關 |

### 視角與顯示設定
| 按鍵 | 功能描述 |
| :--- | :--- |
| **1 ~ 4** | 切換 X、Y、Z 軸平行視角或單一透視視角 |
| **5** | **割四個畫面** (三視圖 + 直升機第一人稱視角) |
| **H / J / K** | 攝影機位移 (前後 / 左右 / 上下) |
| **U / I / Y** | 攝影機旋轉 (Heading / Pitching / Rolling) |
| **X / x** | 視角縮放 (Zoom Out / In) |

## 🛠️ 開發環境
* **語言**: C++
* **圖形庫**: OpenGL (GLUT)
* **外部標頭檔**: `stb_image.h` (用於讀取材質圖片)
* **編譯連結**: 需連結 `opengl32.lib`, `glu32.lib`, `glut32.lib`

## 檔案說明
* `main.cpp`: 程式主邏輯與事件監聽。
* `stb_image.h`: 圖片解碼輔助。
* `image/`: 存放所有貼圖資源 (如 `badge.png`, `sky.png`, `fluff.png` 等)。
